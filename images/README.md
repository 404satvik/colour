# How to supply reference images

Drop an image into **this folder** named after the style's id, e.g. `brutalism.jpg`.
It will automatically appear on that style's card and in its detail popup â€”
no code changes needed. If no file exists, the live CSS demo shows instead.

- **Supported formats:** `.jpg`, `.png`, `.jpeg`, `.webp`, `.avif` (tried in that order)
- **Recommended size:** ~800Ă—500px (landscape). Images are cropped to fill (object-fit: cover).
- **One image per style.** Use the exact filename below.
